Contrôles : Manette de Xbox
Stick gauche : se déplacer
A : dasher
B : parer (s'oriente automatiquement vers l'enemi le plus proche)
X : attaquer (ne s'oriente pas automatiquement)

Note sur le prototype:
-Ce prototype présente ce que sera le tutoriel du jeu (petite partie plateforme pour apprendre à dasher, puis un combat simple)
-Le combat n'est absolument pas finit, en particulier le comportement actuel du Gardien et la mort du joueur)
-Le comportement de l'IA est encore en construction mais l'idée est là : les mobs ne nous fonceront pas dessus tout le temps, ils attendront, selon leur comportement, le bon moment selon eux pour attaquer
